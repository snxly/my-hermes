Gateway — 20+ Messaging Platforms

  • • Built-in Kanban Dispatcher for multi-agent workflows
  • • Pairing system for DM authorization

Tool System — Registry Pattern

  • • One file per tool under tools/ — auto-discovered via registry.register()
  • • Toolsets compose individual tools for different scenarios
  • • check_fn gates tools behind env vars / platform requirements
  • • Narrow waist: every tool shipped on every API call → add sparingly

Skills — Self-Improving Agent

  • • Reusable procedures saved as SKILL.md with YAML frontmatter
  • • Curator: auto-manages lifecycle (stale → archive, never delete)
  • • Skills hub + GitHub tap for community sharing
  • • Preprocessing pipeline: inject into system prompt at session start

Security & Privacy

  • • Secret redaction in tool output (on by default)
  • • Prompt injection scanning on all context files
  • • PII redaction in gateway messages (opt-in)
  • • Command approval: manual / smart (aux-LLM) / off

Profiles & Multi-Tenancy

  • • ~/.hermes/profiles/<name>/ — full isolation per profile
  • • Clone from default or any profile
  • • Kanban multi-profile collaboration with dispatch
  • • Cron, curate, skills, memories all profile-scoped
